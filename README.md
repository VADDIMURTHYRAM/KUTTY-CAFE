# ☕ KUTTY CAFE - India's No.1 Cafe 

An interactive, responsive front-end web application developed as an academic project. Kutty Cafe blends a modern, user-friendly digital cafe experience with the rich flavors of traditional South Indian street food and beverages.

🔗 **Live Demo:** [View Kutty Cafe Live](https://vaddimurthyram.github.io/KUTTY-CAFE/)

---

## 🚀 Key Features

*   **Interactive Menu & Cart System:** Users can browse a curated menu (divided into Beverages, Snacks, and Desserts), add items to a cart, see real-time price calculations, and simulate placing an order.
*   **Dynamic Data Export:** Includes a functional "Download Receipt" feature that exports the user's order details as a `.json` file.
*   **Engaging UI Elements:** Features a creative "Presidential Patrons" testimonial slider and a humorous "Famous Customers" endorsement section.
*   **Customer Support Simulation:** A floating interactive chat widget in the bottom-right corner to mimic real-world e-commerce customer support.
*   **Fully Responsive:** Optimized for a seamless experience across desktops, tablets, and mobile devices.

---

## 🛠️ Tech Stack

*   **HTML5:** Structured the semantic layout of the website.
*   **CSS3:** Handles custom styling, layouts (Flexbox/Grid), and smooth animations.
*   **JavaScript (ES6):** Powers the interactive cart logic, receipt downloads, modal toggles, and sliding testimonials.
*   **GitHub Pages:** Utilized for seamless hosting and deployment.
